# RESULT_LEDGER.md Template

`RESULT_LEDGER.md` records experiment outputs and claim support across attempts.

```markdown
# Result Ledger

| Time | Task | Attempt | Command | Metric | Result | Supports Claim | Logs | Notes |
|---|---|---|---|---|---|---|---|---|
```

Use `Supports Claim` values such as:

```text
yes
partial
no
inconclusive
```

Do not treat implementation completion as evidence for a research claim. Record actual experiment outputs.
