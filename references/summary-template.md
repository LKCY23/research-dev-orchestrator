# SUMMARY.md Template

`SUMMARY.md` is a derived human-readable monitor. It is not a source of truth and may be deleted and regenerated.

Sources of truth remain:

```text
RUN.json
tasks/*/STATUS.json
references/state-machine.json
tasks/*/EVIDENCE.md
RESULT_LEDGER.md
```

Template:

```markdown
# Run Summary

## Objective

## Current Status

## Task Board

## Active Blockers

## Ready For Codex Review

## Protocol Warnings

## Protocol Non-Fatal Warnings

## Recent Decisions

## Recent Events

## Experiment Results

## Next Actions
```
